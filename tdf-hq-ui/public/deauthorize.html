Deauthorization Information

Last updated: 2026-02-12

This endpoint is used for Meta app deauthorization callbacks for TDF InstG Bot.

What deauthorization means

  • Future message access is revoked once app access is removed in Meta.
  • Previously stored conversation data may remain until a deletion request is submitted.

Request data deletion

To request data deletion, visit /data-deletion or email info@tdfrecords.net.
