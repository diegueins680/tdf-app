# Proveedores candidatos con fuentes publicas

Fecha: 2026-04-19

Objetivo: identificar proveedores reales a quienes se puede pedir proforma firmada para soportar el uso de fondos de Domo del Pululahua. Esta lista no implica recomendacion, contrato ni validacion de calidad; solo consolida contactos publicos para iniciar cotizaciones.

## Regla de contacto

No enviar datos privados, documentos bancarios ni informacion sensible del propietario. Para pedir proforma basta enviar el alcance, ubicacion general del proyecto, presupuesto referencial y solicitud de PDF firmado.
