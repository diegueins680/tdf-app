# Revision de preparacion bancaria

Fecha: 2026-04-19

Proyecto: Domo del Pululahua

Monto objetivo: USD 100,000

## Veredicto

El paquete publico seguro esta listo para primer contacto, precalificacion y conversacion inicial con un ejecutivo de banco o cooperativa. Todavia no esta listo para analisis formal de credito porque faltan documentos privados, proformas firmadas, soportes de permisos, estados financieros y respaldo de garantia.

La recomendacion es enviar primero el ZIP publico y usarlo para conseguir un asesor identificado, confirmar producto, plazo, garantia, canal seguro y lista final de requisitos. Solo despues de esa confirmacion se debe enviar la carpeta privada.

## Fortalezas del expediente actual

- El monto solicitado, destino de fondos, modelo operativo y restriccion de cuota estan claros.
- El proyecto no se presenta como idea sin activo fisico: la estructura principal ya esta construida.
- El deudor recomendado es TDF Records como operador, con el propietario como garante si mejora la aprobacion.
- El paquete diferencia documentos publicos de documentos privados.
- La pro forma de 36 meses permite discutir cobertura de deuda, meses negativos iniciales y necesidad de capital de trabajo.
- La pagina publica de cotizacion y reserva ya esta desplegada en `https://tdf-app.pages.dev/domo-del-pululahua`.
- El ZIP incluye evidencia visual del embudo publico de cotizacion y reserva.

## Objeciones probables del banco o cooperativa

| Objecion probable | Riesgo | Respuesta preparada |
| --- | --- | --- |
| El plazo de 18 a 20 anos puede no estar disponible para un credito productivo pequeno. | La cuota podria superar USD 1,000. | Pedir simulacion con garantia real, gracia inicial, menor principal o estructura mixta. |
| Las cifras de uso de fondos son presupuestos de planificacion. | El analista no puede validar costo real. | Reemplazar cada categoria con proforma firmada antes de solicitud formal. |
| Los permisos estan en tramite. | Riesgo operativo antes de eventos grandes. | Entregar estado de tramite, cronograma y plan de operar primero eventos de menor riesgo. |
| No hay estados financieros adjuntos. | No se puede medir capacidad historica de pago. | Entregar declaraciones, estados de cuenta, balances y referencias solo por canal seguro. |
| Falta documentacion de garantia. | El plazo largo puede no justificarse. | Preparar escritura, gravamenes, predial, avaluo y autorizaciones del propietario/conyuge si aplican. |
| La demanda comercial aun no esta probada con reservas pagadas. | Riesgo de ventas iniciales. | Adjuntar leads, cartas de intencion, conversaciones anonimizadas, visitas y calendario de lanzamiento. |

## Recomendacion de envio

1. Enviar `domo-public-lender-packet.zip` solo para precalificacion inicial.
2. Pedir que el caso se analice como credito productivo PYME o productivo empresarial.
3. Pedir al ejecutivo una simulacion para USD 100,000 con cuota objetivo cercana a USD 1,000.
4. Preguntar de forma explicita si la garantia real permite plazo mayor, gracia o mejor tasa.
5. Confirmar si conviene aplicar como TDF Records, propietario persona natural, o estructura con empresa mas garante.
6. Solicitar canal seguro antes de enviar cedula, RUC, estados bancarios, declaraciones, escrituras o avaluos.

## Material que falta para pasar de precalificacion a solicitud formal

- RUC, constitucion, nombramiento, cedula y papeleta del representante legal.
- Declaraciones de impuesto a la renta, IVA, estados bancarios y balances disponibles.
- Escritura, certificado de gravamenes, predial, avaluo y soporte patrimonial del garante.
- Proformas firmadas de paisajismo, camino, mobiliario, senaletica, catering/barra, sonido/luces, seguros, permisos, legal e ingenieria.
- Estado documentado de permisos: uso de suelo, eventos, alimentos, alcohol, sonido, horarios, seguridad, aforo, parqueo, banos y residuos.
- Evidencia comercial privada: leads, conversaciones anonimizadas, cartas de intencion, visitas, fotos profesionales y calendario de lanzamiento.

## Decision recomendada

Avanzar con Pichincha y Produbanco primero, pero no aceptar una aprobacion si la cuota excede la capacidad del modelo. Si ninguna institucion ofrece plazo suficiente para USD 100,000, preparar tres alternativas antes de firmar:

| Alternativa | Uso |
| --- | --- |
| Principal reducido | Mantener cuota cerca de USD 1,000 cuando el plazo maximo sea 10 a 15 anos. |
| Desembolso por tramos | Financiar primero acceso, permisos, mobiliario minimo y capital de trabajo. |
| Mezcla credito + aporte propietario | Cubrir proformas criticas sin forzar una cuota que el flujo inicial no soporte. |
