# Indice privado para canal seguro

Fecha: 2026-04-19

Proyecto: Domo del Pululahua

Uso: armar una carpeta privada fuera de git cuando el prestamista confirme canal seguro.

## Condicion previa

No enviar esta carpeta por WhatsApp publico, bot, correo no verificado ni chat general. Antes de enviar, registrar:

## Orden recomendado de carga

1. Documentos de identidad y empresa.
2. Finanzas y estados de cuenta.
3. Propiedad y garantia.
4. Proformas firmadas y uso de fondos.
5. Permisos y plan operativo.
6. Evidencia comercial privada.
7. Paquete publico ya enviado como referencia.

## Control final antes de enviar

- El asesor y canal fueron confirmados por telefono o agencia.
- Cada archivo tiene nombre claro y fecha.
- Ningun documento se envia por bot o chat publico.
- El manifiesto de carga queda actualizado.
- Se guarda copia del envio y acuse de recibo.
