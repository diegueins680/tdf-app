# Domo del Pululahua tabla de bancos y cooperativas

Fecha: 2026-04-18

Objetivo: comparar bancos y cooperativas para un crédito productivo de USD 100,000, con cuota objetivo máxima de USD 1,000 mensual y plazo lo más largo posible.

Nota: las cuotas son referenciales, no incluyen seguros, comisiones, SOLCA, avalúos, gastos legales ni cambios de tasa. Se calcularon para mostrar el problema central: los plazos públicos comunes de 48 a 84 meses no alcanzan la cuota objetivo, por lo que la negociación debe enfocarse en plazo, gracia, garantía, estructura y capacidad de pago.

## Meta de negociación

| Variable | Meta |
| --- | --- |
| Monto | USD 100,000 |
| Tipo | Productivo PYMES o productivo empresarial, no consumo |
| Cuota máxima | USD 1,000 mensual |
| Plazo ideal | 18 a 20 años |
| Garantía posible | Personal del propietario y/o real sobre inmueble |
| Deudor preferido | TDF Records |
| Deudor alternativo | Propietario, si mejora aprobación o plazo |
| Uso de fondos | Paisajismo, camino, mobiliario, señalética, capital de trabajo, catering/barra, sonido/luces, permisos |

## Preguntas para cada ejecutivo

1. ¿Pueden tratar la solicitud como crédito productivo PYMES o productivo empresarial?
2. ¿Cuál es el plazo máximo real para USD 100,000 con garantía real?
3. ¿Pueden ofrecer gracia de capital mientras se terminan permisos, camino, mobiliario y lanzamiento?
4. ¿La garantía puede ser el inmueble donde opera Domo?
5. ¿Exigen que el deudor sea TDF Records, persona natural o ambos?
6. ¿Qué tasa efectiva anual, seguros, comisiones y gastos de avalúo aplican?
7. ¿Permiten prepago parcial o total sin penalidad?
8. ¿Pueden ajustar pagos a estacionalidad de eventos?
9. ¿Qué documentos revisan antes de emitir una precalificación?
10. ¿Qué monto máximo aprobarían manteniendo la cuota cerca de USD 1,000?

## Mensaje inicial para enviar

Hola, busco evaluar un crédito productivo para TDF Records por USD 100,000 destinado a terminar la puesta en marcha de Domo del Pululahua, un venue de eventos ya construido. El inmueble es propio y podemos conversar sobre garantía personal o real si ayuda a mejorar plazo y cuota.

El objetivo es mantener la cuota cerca de USD 1,000 mensual, por lo que necesitamos revisar el plazo máximo disponible, periodo de gracia, tasa efectiva, garantías y requisitos. Tenemos resumen del proyecto, uso de fondos, flujo proyectado, página de reservas y checklist documental. ¿Con qué ejecutivo puedo revisar una precalificación?
