# Domo del Pululahua resumen para financiamiento

Fecha: 2026-04-18

## Solicitud

TDF Records solicita evaluar un financiamiento productivo de USD 100,000 para terminar la puesta en marcha comercial de Domo del Pululahua como venue para eventos privados, corporativos, culturales y de bienestar.

Objetivo de cuota: hasta USD 1,000 mensuales.

Estructura solicitada: el plazo más largo posible, idealmente 18 a 20 años, con opción de prepago. Si el plazo disponible es menor, se solicita evaluar periodo de gracia, pago flexible por estacionalidad, garantía hipotecaria, reducción de principal o una estructura con saldo residual que mantenga la cuota dentro de la capacidad del proyecto.

## Solicitante propuesto

Primera opción: TDF Records como deudor operativo, con garantía personal del propietario y garantía real sobre el inmueble si el banco o cooperativa lo requiere.

Razón: el destino del crédito es productivo, la operación y los flujos se manejarán desde la empresa, y la propiedad del Domo puede fortalecer el respaldo patrimonial. Se evaluará solicitud personal solo si mejora de forma clara la aprobación, tasa, plazo o garantía.

## Proyecto

Domo del Pululahua es un espacio de eventos ubicado en la zona de Pululahua, cerca de Quito. El Domo tiene diámetro confirmado de 16 m, equivalente a aproximadamente 201 m2 de área circular bruta.

El espacio ya está construido. El financiamiento se usará para completar las partidas necesarias para operar y vender eventos de forma profesional:

| Uso de fondos | Monto |
| --- | ---: |
| Paisajismo y acabados exteriores | USD 20,000 |
| Camino, acceso y flujo de parqueo | USD 16,000 |
| Mobiliario y equipo operativo | USD 18,000 |
| Señalética, contenido y material comercial | USD 6,000 |
| Kit inicial de catering y barra | USD 9,000 |
| Base de sonido e iluminación | USD 7,000 |
| Permisos, legal, seguros e ingeniería | USD 6,000 |
| Capital de trabajo | USD 13,000 |
| Contingencia | USD 5,000 |
| Total | USD 100,000 |

## Modelo de ingresos

Ingresos principales:

- Alquiler de venue para bodas y celebraciones.
- Eventos corporativos, workshops y activaciones de marca.
- Retiros, bienestar y experiencias privadas.
- Conciertos íntimos, sesiones audiovisuales y contenido.
- Sesiones fotográficas y producciones.

Servicios directos operados por Domo:

- Catering.
- Barra.
- Sonido.
- Iluminación.

Servicios tercerizados o coordinados con proveedores:

- Transporte.
- Carpas o estructuras especiales.
- Seguridad adicional.
- Hospedaje aliado.
- Producción especial.

## Capacidad y ventas

Capacidad de planificación antes de permisos finales:

- 70 a 100 invitados sentados para eventos premium.
- 60 a 90 personas para workshops o retiros.
- 120 a 160 personas para formato cóctel, conciertos o actividades de pie, sujeto a permisos, baños, salidas, parqueo y plan de seguridad.

Meta comercial base: 4 eventos mensuales con ticket promedio de USD 5,000 y margen de contribución de 50%.

Punto de equilibrio operativo: aproximadamente 2 eventos fuertes por mes si los costos fijos se mantienen cerca de USD 4,000 y la deuda no supera USD 1,000 mensuales.

El formulario permite estimar un quote, registrar datos del cliente, evento, fecha, invitados y servicios requeridos. La solicitud entra por el API público de reservas de TDF para seguimiento comercial.

## Riesgos y mitigación

| Riesgo | Mitigación |
| --- | --- |
| Permisos en trámite | Avanzar licencias, uso de suelo, seguridad, alcohol, alimentos, sonido y horarios antes de eventos grandes. |
| Estacionalidad | Diversificar bodas, empresas, workshops, producción audiovisual y retiros. |
| Acceso y clima | Invertir en camino, señalética, drenaje, parqueo y proveedores de transporte. |
| Capacidad real | Definir aforo final con permisos, baños, salidas, parqueo y plan de seguridad. |
| Cuota financiera | Negociar plazo, gracia, garantía real y prepago para mantener cuota hasta USD 1,000. |
