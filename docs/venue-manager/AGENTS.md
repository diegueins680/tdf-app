# Instrucciones para `venue-manager`

- Todo material preparado para solicitar, respaldar, negociar o dar seguimiento al prestamo de Domo del Pululahua debe estar en espanol.
- El idioma debe adaptarse a Ecuador: bancos, cooperativas, permisos, garantias, RUC, cedula, declaraciones, escrituras y canales locales.
- Se puede investigar o usar fuentes en ingles, pero los entregables finales para prestamistas deben quedar traducidos y localizados al espanol ecuatoriano.
- Antes de regenerar PDFs o ZIPs para bancos/cooperativas, revisar que no queden encabezados, notas operativas, tablas o instrucciones internas en ingles.
- No incluir documentos privados en git. Cedulas, RUC, declaraciones, estados bancarios, escrituras, garantias, contratos firmados y datos de clientes deben circular solo por canales seguros confirmados por el prestamista.
