# Archives

This directory contains historical files, backups, and one-off artifacts that are kept for reference but are not part of the active codebase.

## Contents

- `epk_bundle/` — Historical EPK (Electronic Press Kit) templates and samples
- `tdf-hq-ui-prev-3/` — Previous iteration of the web UI (backup)
- `tdf-root-ui-legacy/` — Dead root-level React frontend (`src/`, `index.html`, etc.) that was superseded by `tdf-hq-ui/`
- `patches/` — Historical Git patches (applied and preserved for reference)

> These files are **not** built, deployed, or referenced by CI. They exist only for historical context.
